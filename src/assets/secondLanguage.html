第二语言中文班

第二语言中文班共分6级，完成730个生字，1176个词语，144个基本句子的学习。 根据少年儿童和青少年的生理、心理特点和他们的需求与爱好，力求教学内容的科学性、趣味性和实用性。

  1. 寓教于乐，讲求内容的科学性、趣味性和实用性.

    2. “寓教于乐”的理念是少儿汉语教学的灵魂，为了激发少年儿童的想象力、思考力和创造力，帮助他们达到能初步运用汉语进行听说读写的学习目标，我們设计了相应的教学活动，选取贴近他们学习、生活、兴趣的题材，运用对话、采访、节目主持、儿歌、游戏、折纸、角色扮演、讲故事、事物演示和室外教学等多种方式，使他们对汉字和汉语产生兴趣，在教材创设的各种日常学习、生活和游戏的语言场景中轻松、愉快地学习汉语。

  2. 贯彻结构、功能、文化相结合。

    3. 为了使学生具备初步的汉语普通话听说读写的语言交际技能，为进一步学习汉语和了解中华文化打下良好的基础，我们从内容到形式都注意贯彻“以学生为主体，教师为主导”“培养言语交际能力，听说读写全面发展”的教学理念。我们不教语法，而注重基本句型的教学，字词句反复练习，让少儿在对大量语言现象形成感性认识的基础上，逐步领会汉语的遣词造句规则，并让他们既能听懂汉语，会说汉语，又对中国文化有了初步的了解。

  3. 重视汉字教学，展现汉语的独特风貌。

    4. 遵循少儿的认知规律和汉字结构规律，从汉字的初始写法入手，由独体字到合体字，按笔画、笔顺、部首的顺序，先易后难，充分展示汉字形、音、意相结合的特点，努力把汉字教学与汉语教学结合起来，使学生觉得汉字有趣，学得会，写得好；同时，使他们能真切地感受到汉语和中华文化的魅力，从此而激发起他们学习汉语的兴趣。

  4. 汉语拼音是有效的工具。

    5. 汉语拼音是海外少儿学习汉语有效的工具。将汉语拼音的教学与字、词、短语、句子的教学紧密地结合在一起，以求达到既能读准声母、韵母、声调和整体认读音节，又能准确地拼读音节；既学会了汉语拼音，又学习了说话的学习目的。
